## Python和外部库
执行源代码需要安装以下软件。

* Python 3.x
* NumPy
* Matplotlib
 
另外，作为可选项，使用以下库。

* SciPy（可选）
* CuPy（可选）

## 执行方法

移动到各章的文件夹，执行Python命令。

```
$ cd ch01
$ python train.py

$ cd ../ch05
$ python train_custom_loop.py
```
